# The Digital Gallery — Landscape Art Photography

A cinematic photography portfolio featuring scroll-driven video backgrounds, curated galleries, a YouTube learning hub, and workshop booking.

**Live site:** [beckdotan.github.io/dotcomphoto_website](https://beckdotan.github.io/dotcomphoto_website)

## Pages

- **Home** — Cinematic hero with scroll-driven video playback, animated stats, and selected work
- **Gallery** — Category-based portfolio (Night, Landscape, Intimate Landscape, People in Nature) with masonry grids and lightbox
- **Learning** — YouTube video library with tag filters, workshops, and private workshop inquiry form
- **About** — Philosophy, journey, gear, and mission

## Tech Stack

- React 19 (Create React App)
- Tailwind CSS via CDN
- Scroll-driven video frame playback (canvas)
- Cloudinary for image hosting
- Formsubmit for contact form
- GitHub Pages for deployment

## Development

```bash
npm install
node node_modules/react-scripts/scripts/start.js
```

## Deployment

```bash
npm run deploy
```

Deploys to GitHub Pages via `gh-pages`.
